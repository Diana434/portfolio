В этой папке лежат файлы, относящиеся к созданию телеграм бота, для которого в ходе работы была создана база данных с помощью парсинга сайтов, а также производились SQL запросы в эту базу

файл main.py – создание базы данных с помощью парсинга сайта с текстами песен
файл bot.py – запуск бота
файл songbase.py - sql запросы в базу
